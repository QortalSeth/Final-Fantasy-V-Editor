Action data
___________

- D1/0B80-137F: Magic           [Magic]
- D1/59E0-5CDF: Abilities       [Ability] (00-5F)
- D1/6AB1-6DF8: Ability effects [Ability effect]

Action data: 8 bytes
0 Targeting
1 [Attack type]
2 Misc.
3 Reflectable + MP cost
4 Unavoidable + Attack formula
5 Parameter 1
6 Parameter 2
7 Parameter 3

________________________

0: Targeting

0 0 0 0   0 0 0 0
M A T S   E R    

80 Multi-target optional
40 Hits all targets
20 Target selectable
10 Side selectable
08 Target enemy (by default)
04 Roulette
02 
01 


2: Misc.

0 0 0 0   0 0 0 0
M L       { Hits }

80 Monster bit ?
40 Learnable
20 
10 
01-08 Number of hits (minus 1) (Meteo)


3: Reflectable + MP cost

0{0 0 0   0 0 0 0}
nR    MP cost

80 Not reflectable
40-01 MP cost


4: Unavoidable + Attack formula

0 0 0 0   0 0 0 0
U {   Formula   }

80 Unavoidable
40-01 [Attack formula]
